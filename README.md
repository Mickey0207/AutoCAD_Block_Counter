# AutoCAD_Block_Counter

## Version 1.0.0
已完成初期功能,能夠於 AutoCad 中完成圖塊計數功能，並且輸出至命令列表。

## Version 2.0.0
將圖塊計數結果輸出至 Excel 檔案中。

## Version 2.1
可將某一資料夾內的所有 DWG 進行圖塊計數，將結果輸出至 Excel 檔案中。

---
## 版本相容性
- AutoCAD 2025
- AutoCAD 2024
- Framework : .Net 8.0
- OS : Windows 11
